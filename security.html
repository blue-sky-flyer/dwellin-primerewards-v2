Data Security

How we protect your home maintenance records

Last updated: February 2024

Data Security at Dwellin

At Dwellin, protecting your data is our top priority. Your home maintenance records and personal information are critical to your home's care and insurance protection.

Encryption

In Transit

All data transmitted between your device and Dwellin servers is encrypted using industry-standard TLS 1.2+ (Transport Layer Security).

At Rest

Your stored data is encrypted using AES-256 encryption (military-grade). Even if someone gained access to our servers, they could not read your data without the encryption keys.

Authentication & Access Control

Secure Login

  • Secure password hashing using industry-standard algorithms
  • Optional two-factor authentication (2FA)
  • Secure session management
  • Automatic logoff after periods of inactivity

Employee Access

Dwellin employees only have access to data necessary for their specific job functions. We enforce role-based access control, minimum privilege access, and regular access audits.

Infrastructure Security

Cloud Hosting

Dwellin uses Amazon Web Services (AWS) for secure, reliable hosting with:

  • Multiple data center redundancy
  • Automatic backups and disaster recovery
  • DDoS protection and mitigation
  • SOC 2 Type II compliance

Network Security

  • Firewalls and intrusion detection systems
  • DDoS mitigation and rate limiting
  • Web application firewalls
  • Regular network penetration testing

Application Security

Secure Development

  • Secure coding practices
  • Code review process for all changes
  • Dependency scanning for vulnerable libraries

Testing & Scanning

  • Regular security scanning and vulnerability assessments
  • Annual third-party penetration testing
  • OWASP Top 10 vulnerability testing

Compliance & Standards

Dwellin adheres to industry-leading security standards:

  • GDPR (General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)
  • SOC 2 compliance
  • OWASP best practices

Data Backup & Recovery

Backups

Your data is backed up continuously and automatically, multiple times per day, in geographically redundant locations using encrypted backup systems.

Disaster Recovery

We have documented disaster recovery procedures with commitment to restore service within 24 hours in case of emergency.

Incident Response

Detection & Notification

If a security incident occurs, we will:

  • Immediately investigate and contain the incident
  • Assess the scope and impact
  • Notify affected users as required by law
  • Provide clear information and recommended actions

Your Role in Security

You can help protect your account by:

  • Using strong, unique passwords
  • Enabling two-factor authentication
  • Keeping your device and OS updated
  • Avoiding public WiFi for sensitive activities
  • Reporting suspicious activity immediately

Reporting Security Issues

If you discover a security vulnerability, please email security@dwellin.com. We appreciate responsible disclosure and will respond within 24 hours.
